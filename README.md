# 🏥 Patient Distribution Analysis Dashboard

## 📌 Project Overview

This project analyzes a healthcare patient dataset using **Microsoft Excel** and **Tableau Public**.

The dashboard helps visualize:

- Patient distribution by department
- Gender distribution
- City-wise patient analysis
- Monthly admissions
- Revenue by department
- Bed utilization
- Insurance coverage
- Patient outcomes
- Age group distribution

---

## 🛠 Tools Used

- Microsoft Excel
- Tableau Public

## 📊 Dataset

- 500 Patient Records
- Patient Demographics
- Disease Information
- Departments
- Treatment Cost
- Insurance Status
- Bed Type
- Admission & Discharge Dates
- Patient Outcomes
